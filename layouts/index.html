MacOS

Download custom Homebrew formulae and use it to install cairo-lang: 
curl -o /tmp/cairo-lang.rb https://raw.githubusercontent.com/lambdaclass/cairo-by-example/main/homebrew/cairo-lang.rb && brew install --formula --build-from-source /tmp/cairo-lang.rb

Arch Linux

Download the package with your favourite AUR helper, such as: 
yay -S cairo-lang

Other Linux

Requirements: git, rustup 
curl -L https://github.com/franalgaba/cairo-installer/raw/main/bin/cairo-installer | bash
